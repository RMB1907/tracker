Understanding Blockchain

Blockchain, at its core, is a technology that innovates how we transfer value. It operates as a decentralized ledger where:

  • Each transaction is approved by a node and added to the ledger as a new record.
  • The ledger forms a chain, similar to a linked list, where each block is linked using cryptographic hashes.
  • Any change to a node can invalidate the chain, ensuring data integrity.

The term "ledger" reminded me of my Business Economics and Financial Analysis (BEFA) class. Unlike traditional ledgers, blockchain connects transactional data across Web3 rather than a few isolated databases.

Consensus Protocol

Each node in a blockchain network runs software that provides instructions on how to:

  • Verify transactions
  • Add new blocks to the chain

These instructions are collectively referred to as the “consensus protocol.”
