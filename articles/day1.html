Development Environment Requirements

To start with Web3 development, ensure you have the following tools:

  • WSL2
  • Git (Version 2.45.2.windows.1)
  • Docker (Version 4.10+)
  • Python (Version 3.12.0)
  • pipx (Version 1.4.3)
  • VS Code

Note: AlgoKit uses Docker for localnet management and development environment setup.

Getting Started with AlgoKit

Initialize your project with AlgoKit to set up a virtual environment and recommended extensions. Key features include:

  • High-Level Language (HLL) to Low-Level Language (LLL) conversion
  • JSON artifacts stored in the project subdirectory
  • Integration with Dappflow for dApp interaction

Use the following commands to manage your local environment:

  • localnet start
  • localnet status
  • localnet explore
